# 安全安裝

若要針對管理員和店面使用安全通訊端層(SSL) （也稱為HTTPS），您必須設定下列所有引數：

* `--use-secure`：設定為`1`
* `--base-url-secure`：設定為安全URL （亦即，從`https://`開始）
* `--use-secure-admin`設定為`1`

有關這些引數的更多詳細資訊，請參閱本主題稍後的章節。

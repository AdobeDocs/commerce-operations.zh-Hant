# 安全安裝

要對管理員和店面使用安全套接字層(SSL)（也稱為HTTPS），必須設定以下所有參數：

* `--use-secure`:設為 `1`
* `--base-url-secure`:設為安全URL(即從 `https://`)
* `--use-secure-admin` 設為 `1`

有關這些參數的詳細資訊，請參閱本主題稍後的內容。

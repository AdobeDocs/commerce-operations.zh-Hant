# 項目啟動

維護一個健康的電子商務網站對您的業務成功至關重要，這與您最初的發佈一樣重要，其中一個關鍵方面是主動規劃和準備升級，以幫助您的網站盡可能順利地運行。

在此期間未設定正確的升級期望值會導致將來不滿。 確保您瞭解並理解 [釋放](https://devdocs.magento.com/release/)，並使用本指南幫助您定義如何以及何時計畫升級。

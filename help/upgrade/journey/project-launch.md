# 專案啟動

維持一個健全的電子商務網站，對於您的業務成功與初次啟動一樣重要，而其關鍵在於主動規劃及準備升級，以協助您的網站儘可能順暢地運作。

在此期間若未設定正確的升級期望，會導致未來的不滿意。 請確定您瞭解並瞭解 [發行節奏](https://devdocs.magento.com/release/)，並使用本指南來協助您定義如何及何時規劃升級。

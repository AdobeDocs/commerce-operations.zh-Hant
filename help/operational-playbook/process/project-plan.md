# 專案計畫

專案若要獲得成功，不僅需要擁有詳盡的專案計畫，而且作為先決條件，組織完成其業務計畫時務必要熟悉其產品、服務、定價模式、市場、客戶及其地理位置。沒有這份草擬專案計畫將會很困難。

專案計畫是專案團隊、產品所有者和利害關係人同意的初始檔案。 專案計畫應包括下列內容：

- 專案願景
- 目標
- 需求
- 時間表
- KPI
- 團隊
- 里程碑
- 預算
- 風險
- 範圍

一個撰寫良好的專案計畫就像一個藍圖，每個人都需要遵循它才能獲得所需的結果。

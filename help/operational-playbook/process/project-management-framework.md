# 專案管理架構

DACI模型是專案管理架構，用來明確定義專案中不同利害關係人的角色和責任，進而產生有效且高效的群組決策。 DACI代表「驅動程式」、「核准者」、「貢獻者」和「資訊」。 這些角色可清楚說明誰在特定領域和情況下擁有許可權，讓專案順利進行。

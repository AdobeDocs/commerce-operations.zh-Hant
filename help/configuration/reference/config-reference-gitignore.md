# .gitignore引用

Magento Open Source包括底座 `.gitignore` 的子菜單。 請參閱 [最新商務 `.gitignore`](https://raw.githubusercontent.com/magento/magento2/2.4/.gitignore) 的子菜單。 如果必須添加位於 `.gitignore` 清單，您可以 `-f` （強制）選項：

```bash
git add <path/filename> -f
```

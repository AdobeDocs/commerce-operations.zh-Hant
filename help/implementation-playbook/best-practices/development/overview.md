# 開發階段

開發階段包括下列活動：

- 本機和中繼環境設定
- 衝刺計畫
- 票證執行
- 疑難排解
- 程式碼檢閱、合併和測試
- 衝刺檢閱
- 客戶簽署

>[!TIP]
>
>另請參閱 [一般最佳實務](general.md) 以取得有關開發程式整體管理的高階建議。

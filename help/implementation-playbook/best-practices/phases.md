# 實作最佳實務

實作Adobe Commerce專案有許多方式。 本節提供的最佳實務說明Adobe預期客戶和合作夥伴實作特定使用案例的方式。 這些建議旨在幫助防止與站點配置和定制、部署、操作、效能和升級相關的常見問題和潛在問題。

這些最佳實務會依實作階段組織，讓您找到適用於您使用案例的建議。

# Adobe Commerce 2.4.4-p8發行說明

Adobe Commerce 2.4.4-p8是安全性版本，提供三種安全性修正，可增強您的Adobe Commerce 2.4.4或Magento Open Source 2.4.4部署。 它針對先前版本中發現的漏洞進行修正。

{{bics}}

另請參閱 [Adobe安全性公告](https://helpx.adobe.com/security/products/magento/apsb24-18.html) 以取得這些已修正問題的最新討論。


## 安裝和升級指示

如需下載及升級至安全性修補程式發行版本的指示，請參閱 [快速入門安裝](../../../installation/composer.md).

## 更多資訊？

# ACSD-68759：當店面上可見[!UICONTROL Date of Birth]時，客戶帳戶建立失敗，使用阿拉伯語（沙烏地阿拉伯）區域設定

ACSD-68759修補程式修正當&#x200B;**[!UICONTROL Create an Account]**&#x200B;設定為&#x200B;**[!UICONTROL Locale]**&#x200B;阿拉伯文（沙烏地阿拉伯）*且*&#x200B;屬性顯示在店面時，客戶&#x200B;**[!UICONTROL Date of Birth]**&#x200B;頁面無法載入的問題。 安裝[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.77時，即可使用此修補程式。 修補程式ID為ACSD-68759。 請注意，此問題已排程在Adobe Commerce 2.4.9中修正。

## 受影響的產品和版本

**已為Adobe Commerce版本建立修補程式：**

* Adobe Commerce （所有部署方法） 2.4.7-p5

**與Adobe Commerce版本相容：**

* Adobe Commerce （所有部署方法） 2.4.4-p2 - 2.4.4-p16， 2.4.5-p1 - 2.4.8-p3

>[!NOTE]
>
>該修補程式可能適用於具有新[!DNL Quality Patches Tool]版本的其他版本。 若要檢查修補程式是否與您的Adobe Commerce版本相容，請將`magento/quality-patches`包更新為最新版本，並檢查[[!DNL Quality Patches Tool]：搜索修補程式頁](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html)上的相容性。 使用修補程式ID作為搜索關鍵字來查找修補程式。

## 問題

當&#x200B;**[!UICONTROL Create an Account]**&#x200B;設定為阿拉伯文（沙烏地阿拉伯）且&#x200B;**[!UICONTROL Locale]**&#x200B;屬性可見於店面時，**[!UICONTROL Date of Birth]**&#x200B;頁面無法載入，導致`TypeError`。

<u>要再現的步驟</u>：

1. 登入管理員。
2. 前往&#x200B;**[!UICONTROL Stores]** > *[!UICONTROL Attributes]* > **[!UICONTROL Customer]**。
3. 編輯&#x200B;**[!UICONTROL dob]**&#x200B;屬性代碼。
   * 將&#x200B;**[!UICONTROL Show on Storefront]**&#x200B;設為&#x200B;*是*。
4. 按一下&#x200B;**[!UICONTROL Save Attribute]**。
5. 前往「**[!UICONTROL Stores]** > *[!UICONTROL Settings]* > **[!UICONTROL Configuration]** > **[!UICONTROL General]** > **[!UICONTROL Locale Options]**」。
   * 將&#x200B;**[!UICONTROL Locale]**&#x200B;設定為&#x200B;*阿拉伯語（沙烏地阿拉伯）*。
6. 去店面。
7. 開啟&#x200B;**[!UICONTROL Create an Account]**&#x200B;頁。

<u>預期結果</u>：

當&#x200B;**[!UICONTROL Create an Account]**&#x200B;設定為&#x200B;**[!UICONTROL Locale]**&#x200B;阿拉伯文（沙烏地阿拉伯）*且*&#x200B;屬性顯示在店面時，**[!UICONTROL Date of Birth]**&#x200B;頁面載入且沒有錯誤。

<u>實際結果</u>：

* **[!UICONTROL Create an Account]**&#x200B;頁面未載入。
* 會記錄下列錯誤：

  ```
  [2025-10-31T02:14:35.029229+00:00] report.CRITICAL: TypeError: iterator_to_array(): Argument #1 ($iterator) must be of type Traversable|array, null given in app/code/Magento/Customer/Block/Widget/Dob.php:421
  ```

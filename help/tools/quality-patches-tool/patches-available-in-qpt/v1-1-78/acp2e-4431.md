# ACP2E-4431：在重新索引程式期間刪除了符合目標規則的[!UICONTROL Related Products]

ACP2E-4431修補程式修正了在重新索引程式期間刪除與目標規則相符的&#x200B;**[!UICONTROL Related Products]**&#x200B;的問題。 安裝[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.78時，即可使用此修補程式。 修補程式ID為ACP2E-4431。 請注意，此問題已排程在Adobe Commerce 2.4.9中修正。

## 受影響的產品和版本

**已為Adobe Commerce版本建立修補程式：**

* Adobe Commerce （所有部署方法） 2.4.7-p4

**與Adobe Commerce版本相容：**

* Adobe Commerce （所有部署方法） 2.4.4 - 2.4.8-p4

>[!NOTE]
>
>此修補程式可能適用於發行版本為[!DNL Quality Patches Tool]的其他版本。 若要檢查修補程式是否與您的Adobe Commerce版本相容，請將`magento/quality-patches`套件更新至最新版本，並在[[!DNL Quality Patches Tool]上檢查相容性：搜尋修補程式頁面](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=zh-Hant)。 使用修補程式ID作為搜尋關鍵字，以尋找修補程式。

## 問題

在重新索引程式期間，會刪除與目標規則相符的&#x200B;**[!UICONTROL Related Products]**。

<u>要再現的步驟</u>：

1. 建立10000或多個產品。
1. 將10000或多個產品指派至5000或多個類別。
1. 建立100個或更多相關的產品規則。
1. 設定符合使用&#x200B;**[!UICONTROL Product Category]**&#x200B;的&#x200B;**[!UICONTROL Products]**&#x200B;為&#x200B;*X*&#x200B;的規則條件。
1. 使用&#x200B;**[!UICONTROL Product Category]**&#x200B;設定&#x200B;**[!UICONTROL Products]**&#x200B;為&#x200B;**[!UICONTROL Constant Value]** *Y*。
1. 執行此GraphQL查詢以擷取產品的`related_products`：

   ```graphql
   {
       products(filter: { url_key: { eq: "simple-product-491" } }) {
           items {
               sku
               related_products {
                   uid
                   sku
                   name
                   url_key
                   stock_status
                   price_range {
                       minimum_price {
                           regular_price {
                              value
                           }
                           final_price {
                               value
                           }
                       }
                   }
               }
           }
       }
   }
   ```

1. 為`targetrule_product_rule`索引建立大型待處理專案。
1. 為`targetrule_rule_product`索引建立大型待處理專案。
1. 執行`targetrule_rule_product`索引子。
1. 執行`targetrule_product_rule`索引子。

<u>預期結果</u>：

`related_products`清單一律會填入符合規則條件的產品。

<u>實際結果</u>：

當索引子執行時，`related_products`清單會傳回空白陣列。

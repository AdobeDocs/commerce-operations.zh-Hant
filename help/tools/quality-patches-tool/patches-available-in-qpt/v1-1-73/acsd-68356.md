# ACSD-68356： [!DNL GraphQL]購物車查詢傳回不正確的虛擬報價折扣金額

ACSD-68356修補程式修正了[!DNL GraphQL]購物車查詢傳回虛擬報價錯誤折扣金額的問題。 安裝[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.73時，即可使用此修補程式。 修補程式ID為ACSD-68356。 請注意，此問題已排程在Adobe Commerce 2.4.9中修正。

## 受影響的產品和版本

**已為Adobe Commerce版本建立修補程式：**

* Adobe Commerce （所有部署方法） 2.4.8-p2

**與Adobe Commerce版本相容：**

* Adobe Commerce （所有部署方法） 2.4.8 - 2.4.8-p3

>[!NOTE]
>
>此修補程式可能適用於發行版本為[!DNL Quality Patches Tool]的其他版本。 若要檢查修補程式是否與您的Adobe Commerce版本相容，請將`magento/quality-patches`套件更新至最新版本，並在[[!DNL Quality Patches Tool]上檢查相容性：搜尋修補程式頁面](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html)。 使用修補程式ID作為搜尋關鍵字，以尋找修補程式。

## 問題

[!DNL GraphQL]個購物車查詢傳回不正確的虛擬報價單折扣金額，將折扣套用至不合格虛擬產品。

<u>要再現的步驟</u>：

1. 建立兩個虛擬產品：
   1. `virtual1` SKU
   1. `virtual2` SKU
1. 建立購物車價格規則：
   1. 設定為自動套用，不含抵用券。
   1. 動作：
   1. 產品價格折扣百分比 — 20%。
   1. 僅適用於SKU為`virtual2`的購物車專案。
1. 透過店面將產品新增至新的空白報價單：
   1. 請先新增`virtual1` SKU。
   1. 新增`virtual2` SKU秒。
1. 使用下列[!DNL GraphQL]查詢取得購物車詳細資料：

   ```
   {
     cart(cart_id: "cart_id") {
       email
       itemsV2 {
         total_count
         items {
           uid
           prices {
             total_item_discount {
               value
             }
             price {
               value
             }
             discounts {
               label
               amount {
                 value
               }
             }
           }
           product {
             name
             sku
           }
           quantity
         },
         page_info {
             page_size
             current_page
             total_pages
         }
       }
       applied_coupons {
         code
       }
       prices {
         discounts {
           amount {
             value
           }
           applied_to
           label
         }
         grand_total {
           value
         }
       }
     }
   }
   ```

1. 檢查&#x200B;**[!UICONTROL cart]** > **[!UICONTROL items]** > **[!UICONTROL prices]** > **[!UICONTROL discounts]**。

<u>預期結果</u>：

* `virtual1` SKU在[!DNL GraphQL]中未列出折扣
* `virtual2` SKU在[!DNL GraphQL]中列出折扣

<u>實際結果</u>：

* `virtual1`和`virtual2`在[!DNL GraphQL]回應中錯誤顯示相同的折扣金額。

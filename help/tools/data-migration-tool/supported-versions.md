# 支援的資料移轉版本

此 [!DNL Data Migration Tool] 支援移轉 _從_ 下列版本：

* Adobe Commerce：
   * 1.11.x
   * 1.12.x
   * 1.13.x
   * 1.14.x
* Magento Open Source：
   * 1.6.x
   * 1.7.x
   * 1.8.x
   * 1.9.x

如果您從Magento Open Source移轉至Adobe Commerce，系統支援下列版本：

* 1.6.x
* 1.7.x
* 1.8.x
* 1.9.x

## 移轉至版本

若要瞭解支援的版本（您要移轉至的版本），請參閱 [!DNL Data Migration Tool]的 [發行頁面](https://github.com/magento/data-migration-tool/releases).

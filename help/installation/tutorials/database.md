# 建立資料庫架構

在運行此命令之前，必須 [建立或更新部署配置](deployment.md)。

## 配置資料庫並添加資料

命令用法：

```bash
bin/magento setup:db-schema:upgrade
```

要查看資料庫的狀態，請輸入

```bash
bin/magento setup:db:status
```
